## Tips
- JPEG at ~80% quality, or WebP if you have it.
- Keep each file under 200KB if possible — total page weight matters on mobile.
- Resize *before* uploading; don't push 4000px phone photos.
- If a file is missing, the striped placeholder shows automatically (graceful fallback).
